
### Seleccion: 

````Java
if (condicion){
	//acciones;
}
else{
	//acciones;
}
````

---
### Iteracion pre-condicional:

````Java
while (condicion){
	//acciones;
}
````

---
### Iteracion post-condicional:

````Java
do{
	//acciones;
} while (condicion)
````

---
### Repeticion:

````Java
for (i=0; i<Valor; i++){
	//acciones;
}
````

*Obs:* La variable "i" existe unicamente dentro del for, fuera de ella no se conoce.
